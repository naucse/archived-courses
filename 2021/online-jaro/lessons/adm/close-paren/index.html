Zapomenutá závorka

Občas se stane, že v interaktivním režimu Pythonu zapomeneš uzavřít závorku:

>>> quit(
...

Python pak se pomocí ... ptá na pokračování příkazu. Python totiž umožňuje rozepsat případný obsah závorek na víc řádků a dokud závorku neukončíš (nebo nevyvoláš chybu), neukončí.

Cesta ven je klávesová zkratka Ctrl+C, která ukončí zadávaný příkaz a nechá tě zadat nový:

>>> quit(
... # (Zmáčknu Ctrl+C)
KeyboardInterrupt
>>>

V „grafických“ programech typicky Ctrl+C znamená kopírovat, ale v příkazové řádce má starší význam: ukončení. Python ukončení pomocí klávesové zkratky nazývá KeyboardInterrupt:
