Instalace

Nejdříve propoj modul s počítačem přes USB kabel, jako kdybys připojovala třeba mobil.

Je potřeba použít kvalitní datový kabel. Nekvalitní kabely (např. spousta kabelů k nabíječkám) jsou často nepoužitelné.

Dál postupuj podle operačního systému na svém počítači:

  • Linux
  • macOS
  • Windows

Kdyby něco nefungovalo, poraď se s koučem. Původní (anglický) návod k této části je na stránkách MicroPythonu.
