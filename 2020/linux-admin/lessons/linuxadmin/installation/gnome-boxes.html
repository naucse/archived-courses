Instalace Gnome Boxes

Gnome Boxes jsou pro Linux k dispozici normálním způsobem, kterým instaluješ a spouštíš grafické („okýnkové“) programy.

Z příkazové řádky to jde takto:

  • Fedora

    $ sudo dnf install gnome-boxes

  • Debian/Ubuntu

    $ sudo apt-get install gnome-boxes

Program pustíš opět stejným způsobem jako grafické programy, případně z řádky:

  $ gnome-boxes
